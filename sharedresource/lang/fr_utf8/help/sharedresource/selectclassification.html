Sélection et configuration des classifications apparentes


Tous les taxons d'une classification sélectionnée apparaîtront dans une liste déroulante dans le formulaire de description des métadonnées si le plugin de classification est activé.

La suppression d'une classification entraîne nécessairement la suppression des métadonnées y faisant référence pour toutes les ressources mutualisées.

Le bouton "Configurer" permet d'accéder à la sélection des taxon path apparents pour la classification choisie.
