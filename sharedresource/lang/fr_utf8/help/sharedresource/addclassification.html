Ajout de classification


Une classification fait référence à une table en base de données, qui doit être constituée de quatre champs :
  • id
  • parent, qui correspond à l'id du père du taxon en question
  • label, qui correspond à la dénomination du taxon
  • ordering, pour donner l'ordre d'importance entre les différents taxons fils. Un ordering minimum est défini pour savoir si le premier fils a un ordering à 0 ou 1.

Le nom de la table en base de données ainsi que le nom des champs correspondant à ces quatre champs sont à indiquer dans le formulaire.
A noter que le nom de la table peut être indiqué avec le préfixe des tables de moodle ou sans, et que le champs ordering est facultatif (dans le cas où il n'est pas indiqué, l'ordre d'apparition est fait suivant l'id).

La configuration de chaque classification ajoutée est ensuite nécessaire via le bouton "Configurer".
