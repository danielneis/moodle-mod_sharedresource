Add a classification


A classification referecences to a database table, which must contains at least these four fields :
  • id
  • parent, which is the id of the taxon's father
  • label, which is the name of the taxon
  • ordering, to give a specific order for children taxons. A minimum ordering is required to know if the first child have a ordering at 0 or 1.

The name of the database tale, and names of the fields must be written in the form.
You have to notice the name of the table can be written with the prefix of the moodle database tables or not, and that the name of the ordering field is optional (in this case, the order of the children taxons are determined by the id).

The configuration of each classification added is then necessary by clicking on the "Configure" button.
